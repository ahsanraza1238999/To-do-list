# To-Do List (Python CLI App)

A simple and interactive command-line **To-Do List manager** built in Python.  
It helps you manage daily tasks, mark them as done, remove them, and track progress with a visual bar.

---

## Features
- 📌 Add tasks  
- ✅ Mark tasks as done  
- ❌ Remove specific tasks  
- 🧹 Clear the entire list  
- 📊 Track completion with a progress bar  

---

## Requirements
- Python 3.x (Windows recommended)  
- `msvcrt` (built-in for Windows)  

---
